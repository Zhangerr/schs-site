Resources for your convenience

HTML Editors

It is highly recommended you use the zen coding plugin for all of the following editors.
Editor Description
Notepad++ (Windows only, free as in "free beer" and "free speech"!)
Sublime Text (Commercial but indefinite free evaluation, available for all OSes [yes even you linux people have a suggestion but the stock gnome one is good enough])
Komodo Edit Another cross-platform editor that has the nice ftp/sftp features of notepad++, and compatible with zen coding.
Note: Any other text editors mentioned on the zen-coding page that it supports is probably half decent too!

Educational

Editor Description
Club Agenda A google calendar on upcoming lessons. Alternatively, you can view the one embedded on the main page.
Mozilla Developer Network (MDN) A resource from the creators of firefox (mozilla) that is pretty decent.
sitepoint An excellent reference for tags and css properties, especially because it gives browser compatibility. Refer to this over w3schools anyday.
mrbuzbee As strange as the title of the website is, this is where I get the lessons for our agenda. Highly worth reading ahead if you have time.
Opera Developer Network Opera is another browser that is decent but has a much smaller presence. They, however, have excellent resources on beginner and intermediate web design.
W3 w3 is the governing body over HTML and CSS standards (plus a bunch of other stuff) so their resources are a bit more heavy but reading the actual specification for a language can be very enlightening at higher levels.
University of Reddit On a slight tangent, University of Reddit has excellent free courses for many subjects but also has many decent web development courses. In fact, the agenda we're following is one of these.

Miscellaneous

Editor Description
Mercurial Tutorial A short tutorial I wrote on mercurial, which I initially intended we would use to collaborate on projects.
Kuler Useful utility made by Adobe that allows easy color paletting (presets and user submitted ones)
jsfiddle Very useful website that allows you to write html snippets and run it in browser. Please submit all work to me using this.
stackoverflow A useful Q&A site that you can use to ask most programming questions (including web design) if I'm not available.
